# PVParser-Release

This repository provides the code artifacts associated with the manuscript *Reverse Engineering Industrial Process Variables Using Search-Based Optimization*. The release serves to document the existence of executable artifacts during the peer-review process.

## Scope of This Release

In accordance with the open-science statement in the manuscript, this repository makes available the parts of PVParser necessary to support artifact verification at the review stage. The complete implementation and full execution examples will be released after the manuscript is formally accepted.

## Data Availability

The experimental evaluation in the manuscript is conducted on the SCADA Network, SWaT, and WADI industrial CPS datasets. These datasets originate from external research testbeds and cannot be publicly redistributed due to access agreements.
