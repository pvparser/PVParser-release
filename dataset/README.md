# Dataset Availability

This repository does not redistribute the SWaT or WADI datasets.

Consistent with the open-science statement in the paper, please request and download these datasets from the official iTrust/SUTD dataset pages:

- SWaT: https://itrust.sutd.edu.sg/itrust-labs_datasets/dataset_info/#swat
- WADI: https://itrust.sutd.edu.sg/itrust-labs_datasets/dataset_info/#wadi

After obtaining access, place the downloaded files under the corresponding local folders:

- `dataset/swat/`
- `dataset/wadi_enip/`
